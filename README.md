
# Calculadora IMC

Calculadora de IMC!

## Funcionalidades ⚙️

- Adicionar peso (kg) e altura (cm).
- Obter o resultado do seu IMC.

## Stack utilizada ⚛️

**Front-end:** HTML, CSS, JavaScript
